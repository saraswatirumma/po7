Here’s a complete, clean, and runnable HTML example that you can use as a starting point for any webpage.
It includes headings, paragraphs, images, links, lists, and a simple table — all with proper structure.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Sample HTML Page">
    <title>My First HTML Page</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 20px;
            background-color: #f9f9f9;
        }
        h1 {
            color: #2c3e50;
        }
        a {
            color: #2980b9;
            text-decoration: none;
        }
        a:hover {
            text-decoration: underline;
        }
        table {
            border-collapse: collapse;
            width: 50%;
            margin-top: 10px;
        }
        table, th, td {
            border: 1px solid #333;
        }
        th, td {
            padding: 8px;
            text-align: left;
        }
    </style>
</head>
<body>

    <h1>Welcome to My Webpage</h1>
    <p>This is a simple HTML page with basic elements.</p>

    <h2>Links</h2>
    <p>Visit <a href="https://www.example.com" target="_blank">Example Website</a> for more info.</p>

    <h2>Image</h2>
    <img src="https://via.placeholder.com/150" alt="Sample Image">

    <h2>List Example</h2>
    <ul>
        <li>HTML</li>
        <li>CSS</li>
        <li>JavaScript</li>
    </ul>

    <h2>Table Example</h2>
    <table>
        <tr>
            <th>Name</th>
            <th>Role</th>
        </tr>
        <tr>
            <td>Alice</td>
            <td>Developer</td>
        </tr>
        <tr>
            <td>Bob</td>
            <td>Designer</td>
        </tr>
    </table>

</body>
</html>

Features in this HTML:
Proper HTML5 structure (<!DOCTYPE html>, <html>, <head>, <body>).
Meta tags for character encoding, viewport, and description.
CSS styling inside <style> for quick customization.
Headings, paragraphs, links, images, lists, and tables.
Responsive design basics with meta viewport.
